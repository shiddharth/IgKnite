A short introduction!

Veron1CA is an open source Discord moderation & music bot designed to do server management and entertain both at once. It has a bunch of easy-to-use commands that can pretty much make the work of customizing your Discord server a breeze!

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of how Discord bots work.
  • Good knowledge on setting up / hosting code on your own.
  • Have some satisfactory knowledge in Discord Server Moderation.
Setting things up...

The first and the very basic step of using Veron1CA in your own server of choice is to add it. To do this, simply go to Veron1CA's official website and click on 'Inject'. This will prompt you to a screen that's identical with the reference image shown above. Simply select the server you wanna add Veron1CA in, do an easy human verification check and you should see Veron1CA in your server.

The next part is to get to know how to actually use it. First of all, create two roles, BotMod and BotAdmin. Here's what they do:

  • BotMod - This role is used for unlocking the server moderation capabilities for any user with this role.
  • BotAdmin - This role can do everything from the previous role and it also unlocks server customizing abilities for any user with this role.
  • You should assign only one of them to each person, as BotAdmin can do whatever BotMod, you don't really need two roles, it just makes it more messy.

Now when it comes to actually using Veron1CA, you don't need any roles to access the commands. But if you're the moderator / admin of the server itself, then you may need to crank your powers a bit higher up, and that's why Veron1CA's moderation features exist. You can assign yourself BotAdmin if you're the admin and BotMod if you're the moderator, but make sure that no roles get assigned to someone by mistake, it can lead to *chaotic* disasters.

Okay, coming to the actual subject, now you can use the built-in commands to get accquainted with the system and stuff. Here is a small explanation on how commands work in Veron1CA:

# The default prefix for Veron1CA is vrn. and you can start by typing your very first command with every Discord bot.
vrn.help

# You can add an 'all' keyword to display all the available commands that one can use.
vrn.help all

# You can also type a command's name instead of the keyword to get help on that particular command.
vrn.help commandname

# The final stage, where we actually use a command. Remember when you saw all the commands in a list? Well, type vrn. followed by the command to use it, the ping command is an example here which shows the bot's status.

vrn.ping

You should now be well on your way into tweaking and playing with the features that Veron1CA gives you out of the box.
